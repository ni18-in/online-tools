À propos de Devine le Logo

Devine le Logo est un jeu amusant et stimulant conçu pour tester vos connaissances sur les marques populaires du monde entier !

Mettez à l'épreuve vos compétences de reconnaissance de marque à travers différents niveaux de difficulté et modes de jeu. Que vous jouiez en solo ou avec des amis, préparez-vous pour une aventure palpitante.

Ce jeu est développé avec HTML, CSS et JavaScript natif, offrant une application web légère et adaptative.

Bon jeu !

Comment jouer

Jeu général : Un logo s'affiche à l'écran. Votre but est de choisir le nom correct de la marque parmi les quatre propositions à choix multiples.

  • Mode Solo : Jouez à votre propre rythme. Marquez des points à chaque bonne réponse. Tentez de réaliser le meilleur score pour figurer au classement !
  • Mode Fête : Idéal pour jouer à plusieurs ou sur un grand écran ! Un chronomètre augmente la tension à chaque question et le jeu passe automatiquement au logo suivant.
  • Mode Défi : Une course contre la montre chronométrée. Devinez le plus de logos possible avant la fin du temps imparti.
  • Mode Streaming : Installez-vous confortablement et observez ! Les logos et leurs réponses s'affichent automatiquement à la suite. Parfait pour un quiz visuel ou pour apprendre à connaître les marques.

Contrôles :

  • Utilisez votre souris ou le tactile pour sélectionner une option ou cliquer sur les boutons.
  • Clavier : Utilisez les touches 1, 2, 3, 4 ou A, B, C, D pour sélectionner l'option correspondante. Appuyez sur Entrée ou la barre d'espace pour les actions principales comme "Lancer le jeu", "Question suivante" ou "Rejouer".

Ajustes : Avant de commencer, vous pouvez personnaliser votre expérience :

  • Choisissez le mode de jeu de votre choix.
  • Sélectionnez la difficulté (Facile, Moyen, Difficile).
  • Définissez le nombre de manches (pour les modes Solo et Fête) ou la durée de la session (pour le mode Défi).
  • Activez ou désactivez les effets de sonores.

Amusez-vous bien !
