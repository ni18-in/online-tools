Mode d'emploi

Cet outil vous permet de comparer deux objets JSON ou blocs de texte et de visualiser leurs différences.

  • Saisie des données : Collez vos données JSON ou textuelles dans les zones de saisie gauche et droite (avec les numéros de ligne !), ou utilisez les boutons de chargement pour charger à partir de fichiers (.json ou .txt).
  • Mettre en forme (JSON uniquement) : Utilisez le bouton (ou Ctrl+B) pour formater la saisie *s'il s'agit de JSON valide*. Cela ne fonctionnera pas sur du texte brut.
  • Comparer : Cliquez sur le bouton "Comparer" (ou appuyez sur Ctrl+Entrée) pour voir les différences.
  • Résultats : Les différences sont mises en évidence ci-dessous (voir la légende au-dessus des résultats). Vert indique des ajouts, Rouge indique des suppressions, et Bleu indique des modifications.
  • Options (JSON uniquement) : Cochez la case "Ignorer l'ordre des tableaux" si l'ordre des éléments au sein des tableaux n'a pas d'importance pour votre comparaison. Cela ne s'applique que si *les deux* entrées sont du JSON valide.
  • Copier/Télécharger : Utilisez les boutons sous les entrées pour copier les résultats du diff ou télécharger les données brutes du diff (au format JSON, même si les entrées étaient du texte).
  • Mode Dev : Saisissez :dev à la fin de l'une des zones de saisie pour charger des exemples de données JSON.
