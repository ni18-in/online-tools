Comment utiliser

  1. Glissez-déposez votre image (PNG, JPG, WEBP) dans la zone de chargement.
  2. Vérifiez les paramètres. Nous redimensionnons automatiquement les images à la "puissance de 2" la plus proche (par exemple, 512x512) requise par les moteurs de jeu.
  3. Cliquez sur Convertir. L'outil génère le fichier .vtf localement dans votre navigateur.
  4. Téléchargez et enregistrez votre nouveau fichier de texture.

À propos

Ce convertisseur VTF est un utilitaire côté client sécurisé conçu pour les créateurs de mods du moteur Source.

Formats pris en charge :
Entrée : PNG, JPEG, WEBP
Sortie : VTF (7.2, RGBA8888)

Note sur la confidentialité : Vos images sont traitées entièrement dans votre navigateur. Aucun fichier n'est jamais envoyé vers un serveur.

Foire aux questions

Pourquoi mon image est-elle redimensionnée ?

Le moteur Source (format VTF) exige strictement que les dimensions de l'image soient une "puissance de deux" (par exemple, 64, 128, 256, 512, 1024, 2048, 4096). Nous gérons ce redimensionnement automatiquement pour garantir le fonctionnement de votre texture en jeu.

Y a-t-il un fond transparent ?

Oui ! Si votre image PNG d'entrée a de la transparence, elle est préservée dans le fichier VTF final à l'aide du canal Alpha.
