Calculateur de notes universel

Calculez votre moyenne pondérée facilement et rapidement. Les données sont sauvegardées automatiquement.

Mode d'emploi

  1. Définir les catégories : Saisissez un nom de catégorie (ex. Travaux pratiques) et son coefficient/poids (%), puis cliquez sur "Ajouter une catégorie". Il est recommandé que la somme des poids soit égale à 100%.
  2. Ajouter des devoirs : Sous chaque catégorie, utilisez le formulaire pour saisir le nom du devoir, votre note et le total des points possibles. Cliquez sur le bouton "Ajouter" de cette catégorie.
  3. Note en temps réel : Votre note globale calculée est automatiquement mise à jour dans l'encadré Note finale sur le côté droit de l'écran au fur et à mesure que vous saisissez vos notes.
  4. Persistance des données : Vos données sont sauvegardées localement dans votre navigateur. Utilisez le bouton Exporter pour créer un fichier JSON de sauvegarde ou Effacer toutes les données pour réinitialiser complètement le calculateur.

À propos de ce calculateur

Ce calculateur de notes universel est conçu pour fournir un moyen simple et rapide de calculer votre moyenne de cours pondérée. Il est construit en HTML moderne, avec Tailwind CSS pour un design adaptatif, et du JavaScript natif pour des calculs robustes côté client. L'accessibilité a été prise en compte en utilisant des éléments sémantiques et des attributs ARIA pour garantir l'utilisation avec des technologies d'assistance comme les lecteurs d'écran.

Toutes les données sont automatiquement sauvegardées dans le stockage local de votre navigateur.
