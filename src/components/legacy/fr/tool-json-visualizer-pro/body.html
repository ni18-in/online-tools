Comment utiliser JSON Visualizer Pro

Cet outil vous permet de visualiser, de mettre en forme et de gérer facilement vos données JSON.

  • Entrée JSON : Collez ou saisissez vos données JSON dans le panneau de gauche. Les erreurs de syntaxe éventuelles s'afficheront sous la zone de saisie.
  • Visualiser / Mettre à jour : Cliquez sur ce bouton pour analyser la saisie et afficher l'arborescence structurée dans le panneau de droite.
  • Mettre en forme : Nettoie et formate le JSON dans la zone de saisie avec une indentation standard (2 espaces).
  • Minifier : Supprime tous les espaces vides superflus du JSON dans la zone de saisie pour le rendre compact.
  • Copier : Copie le contenu actuel de la zone de saisie JSON dans le presse-papiers.
  • Effacer : Vide la zone de saisie et le panneau de visualisation.
  • Annuler (Ctrl+Z) : Annule la dernière action effectuée (Visualisation/Mise à jour, Mise en forme, Minification, Effacement).
  • Rétablir (Ctrl+Y) : Rétablit la dernière action annulée.
  • Panneau de visualisation : Affiche une arborescence indentée avec coloration syntaxique de vos données JSON valides.

Utilisez les boutons et les raccourcis clavier pour gérer efficacement votre JSON !
