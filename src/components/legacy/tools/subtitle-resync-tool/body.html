How to Resync Subtitles Online?

  1. Upload Video & Subtitles: Select your video file (MP4, WebM, etc.) and the out-of-sync .SRT file.
  2. Check Sync: Play the video to see when the expected dialogue happens versus when the subtitle appears.
  3. Adjust Timing:
    • If subtitles appear too early, click + Shift (Delay).
    • If subtitles appear too late, click - Shift (Advance).
  4. Fine Tune: Use the manual input for precise millisecond adjustments (e.g., -1.5s).
  5. Download: Click "Download Fixed SRT" to save the permanently corrected file.
