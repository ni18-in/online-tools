Cómo usar esta aplicación

  1. Introduce el valor de píxeles (px) deseado en el primer cuadro de entrada.
  2. Ajusta el tamaño de fuente base desde el menú desplegable si tu proyecto usa un valor predeterminado diferente (el estándar es 16px).
  3. El equivalente en REM se calcula y muestra en tiempo real en el cuadro de resultados.
  4. Para una referencia rápida, consulta la tabla de conversión a continuación, basada en un tamaño de fuente base de 16px.
