Cómo Usar

Sigue estos sencillos pasos para grabar tu pantalla:

  1. Ve a Inicio: Asegúrate de estar en la pestaña "Inicio".
  2. Configura las Opciones (Opcional):
    • Marca "Grabar Audio" si deseas incluir audio.
    • Si "Grabar Audio" está marcado, puedes especificar si deseas grabar "Audio del Sistema" (audio de la pestaña/ventana seleccionada) y/o "Micrófono".
  3. Iniciar Grabación: Haz clic en el botón "Iniciar Grabación".
  4. Elige la Pantalla: Tu navegador te pedirá que selecciones lo que deseas compartir: toda la pantalla, una ventana de aplicación específica o una pestaña del navegador. Elige tu preferencia y haz clic en "Compartir".
  5. Controles de Grabación:
    • Una vez iniciada la grabación, verás un temporizador y un indicador visual.
    • Usa "Pausar" para detener temporalmente la grabación.
    • Usa "Reanudar" para continuar una grabación pausada.
    • Usa "Detener Grabación" para finalizar. Se te pedirá confirmación.
  6. Descargar: Después de detener, tu video aparecerá en la sección "Descargas". Haz clic en el botón "Descargar" junto al nombre del archivo para guardar el archivo de video .webm o .mp4.
  7. Permisos: La primera vez que uses la aplicación, tu navegador te pedirá permiso para acceder a tu pantalla y micrófono. Por favor, otorga estos permisos para que la aplicación funcione correctamente.

Nota: Todas las grabaciones se procesan y almacenan localmente en tu navegador. No se sube ningún dato a ningún servidor.

Acerca de Esta Aplicación

Este Grabador de Pantalla es una aplicación web ligera del lado del cliente que te permite grabar fácilmente tu pantalla, ventanas de aplicaciones específicas o pestañas del navegador directamente desde tu navegador web.

Características Principales:

  • Grabación del Lado del Cliente: Todo el procesamiento ocurre en tu navegador. No se envían datos a ningún servidor, garantizando tu privacidad.
  • Múltiples Opciones de Grabación: Elige grabar toda tu pantalla, una sola ventana o una pestaña del navegador.
  • Grabación de Audio: Incluye opcionalmente audio de tu micrófono y/o audio del sistema (de la pestaña/ventana compartida).
  • Controles Sencillos: Funcionalidad fácil de usar para iniciar, pausar, reanudar y detener.
  • Descarga Instantánea: Las grabaciones están disponibles para descargar como archivos .webm o .mp4 inmediatamente después de detener.
  • Material Design: Una interfaz limpia, moderna y responsiva inspirada en Material Design 3 de Google.
  • Modo Oscuro: Alterna entre temas claro y oscuro para una visualización cómoda.

Casos de Uso:

  • Crear tutoriales o demostraciones de software.
  • Grabar videojuegos.
  • Capturar presentaciones o reuniones en línea (asegúrate de tener consentimiento).
  • Reportar errores mostrando visualmente un problema.
  • Guardar videos o transmisiones en línea (respeta los derechos de autor y los términos de servicio).

Esta aplicación utiliza la API de Grabación MediaStream integrada del navegador, lo que la hace eficiente y segura.
