Sobre Adivina el Logotipo

¡Adivina el Logotipo es un juego divertido y desafiante diseñado para poner a prueba tu conocimiento de marcas populares de todo el mundo!

Pon a prueba tus habilidades de reconocimiento de marca en varios niveles de dificultad y modos de juego. Ya sea que juegues solo o con amigos, prepárate para una emocionante aventura de adivinanzas.

Este juego está creado con HTML, CSS y JavaScript nativo, mostrando una aplicación web ligera y adaptativa.

Cómo jugar

Juego general: Se mostrará un logotipo en la pantalla. Tu tarea consiste en elegir el nombre correcto de la marca entre las cuatro opciones de opción múltiple proporcionadas.

  • Modo un jugador: Juega a tu propio ritmo. Gana puntos por cada respuesta correcta. ¡Intenta conseguir la puntuación más alta y entrar en la tabla de clasificación!
  • Modo fiesta: ¡Perfecto para jugar con amigos o en una pantalla más grande! Un temporizador añade emoción a cada pregunta, y el juego avanza automáticamente al siguiente logotipo.
  • Modo desafío: Corre contra el reloj en esta sesión cronometrada. Responde correctamente a tantos logotipos como puedas antes de que se agote el tiempo.
  • Modo streaming: ¡Relájate y aprende! Los logotipos y sus respuestas correctas se mostrarán automáticamente uno tras otro. Este modo es ideal para un cuestionario visual rápido o para familiarizarte con las marcas.

Controles:

  • Usa el ratón o toca para seleccionar una opción o hacer clic en los botones.
  • Teclado: Usa las teclas 1, 2, 3, 4 o A, B, C, D para seleccionar la opción de respuesta correspondiente. Usa Enter o la barra espaciadora para los botones de acción principal como "Iniciar partida", "Siguiente pregunta" o "Jugar de nuevo".

Ajustes: Antes de empezar, puedes personalizar tu experiencia de juego:

  • Elige tu modo de juego preferido.
  • Selecciona la dificultad (Fácil, Medio, Difícil).
  • Establece el número de rondas (para el modo Un jugador/Fiesta) o la duración de la sesión (para el modo Desafío).
  • Activa o desactiva los efectos de sonido.

¡Diviértete adivinando!
