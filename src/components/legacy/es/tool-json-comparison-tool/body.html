Cómo Usar

Esta herramienta te ayuda a comparar dos objetos JSON o bloques de texto y visualizar sus diferencias.

  • Ingresar Datos: Pega tus datos JSON o texto en las áreas de entrada izquierda y derecha (¡con números de línea!), o usa los botones de subir archivos (.json o .txt).
  • Embellecer (solo JSON): Usa el botón (o Ctrl+B) para formatear la entrada *si es JSON válido*. No funciona con texto plano.
  • Comparar: Haz clic en el botón "Comparar" (o presiona Ctrl+Enter) para ver las diferencias.
  • Resultados: Las diferencias se resaltan abajo (ver leyenda sobre los resultados). Verde indica adiciones, Rojo indica eliminaciones, y Azul indica modificaciones.
  • Opciones (solo JSON): Usa la casilla "Ignorar Orden de Arrays" si el orden de los elementos dentro de los arrays no importa para tu comparación. Solo aplica si *ambas* entradas son JSON válido.
  • Copiar/Descargar: Usa los botones debajo de las entradas para copiar los resultados del diff o descargar los datos del diff en bruto (como JSON, incluso si las entradas eran texto).
  • Modo Dev: Escribe :dev al final de cualquier área de entrada para cargar datos JSON de ejemplo.
