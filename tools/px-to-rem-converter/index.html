How to Use This App

  1. Enter the desired pixel (px) value in the first input box.
  2. Adjust the base font size from the dropdown if your project uses a different default (the standard is 16px).
  3. The REM equivalent is calculated and displayed in real-time in the result box.
  4. For quick reference, consult the conversion table below which is based on a 16px base font size.
